  1. Cos'è PHP?

    PHP è un linguaggio di scripting per il web. Vale a dire che tramite di esso potrai realizzare dei piccoli programmi che gireranno sul server nel momento in cui un utente accederà a quella pagina. In questo modo potrai realizzare le cosiddette "pagine dinamiche"

  2. PHP è simile ad ASP?

    Sì, il PHP è simile ad ASP ma ovviamente la sintassi si differenzia. Non stare a sentire le voci: "PHP è meglio di ASP" / "ASP è meglio di PHP". Ognuno ha i suoi gusti e ognuno ha le sue esigenze. Entrambi sono linguaggi sufficientemente forti e robusti per sviluppare applicazioni web di tutto rispetto.

  3. Il mio server supporta PHP?

    Per vedere se il server di cui disponi supporta il linguaggio PHP crea una pagina così:
    <?PHP
phpinfo();
?>
    e uploadala sul server con estensione .php
    Se visualizzandola con il tuo browser otterrai una serie di informazioni allora vuol dire che il tuo server supporta il PHP. Ricordati poi di cancellarla o di proteggerne la visualizzazione con una password, altrimenti chiunque potrà accedere ad informazioni sul tuo server che sarebbe meglio non fornire a nessuno.

  4. Come si visualizza la data?

    Ci sono molti di visualizzare una data in PHP. Uno di questi è il seguente:
    <?PHP
/* visualizza una data del tipo: 31 luglio 2001 */
echo date('H:i:s d/m/Y');
?>

  5. Come si visualizzano le variabili di sistema?

    Per visualizzare le variabili di sistema crea una pagina così:
    <?PHP
phpinfo();
?>
    e uploadala sul server con estensione .php
    Visualizzala con il browser ed avrai tutte le indicazioni di cui hai bisogno e che ti è dato il permesso di vedere

  6. Su quale piattaforma funziona?

    PHP funziona su svariate piattaforme, dai vari ambienti Unix/Linux a macchine Windows. E' sufficiente avere il modulo appropriato installato nel webserver o sulla macchina in questione

  7. Il PHP può essere usato con i database Access?

    Certamente. Il PHP supporta nativamente l'accesso a moltissimi sistemi di database (DBMS). Per vedere le funzioni specifiche ai vari DBMS supportati guarda sul manuale del PHP che trovi sul sito all'indirizzo http://www.php.net/
